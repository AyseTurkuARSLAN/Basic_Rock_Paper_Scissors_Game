# 🪨📜✂️ Taş-Kağıt-Makas (Rock-Paper-Scissors) - Python

Bu basit Python projesi, klasik Taş-Kağıt-Makas oyununu bilgisayara karşı oynamanızı sağlar. Döngüler ve koşullu ifadeler (if-else) mantığını anlamak için harika bir başlangıç seviyesi projesidir.

## 🚀 Özellikler
- Kullanıcıdan alınan girdileri kontrol eder (Hatalı girişleri reddeder).
- `random` kütüphanesi kullanarak bilgisayarın rastgele seçim yapmasını sağlar.
- Oyun sonunda skor sonucunu ekrana yazdırır.
- Kullanıcıya oyunu bitirmek isteyip istemediğini soran bir döngü yapısına sahiptir.

## 🛠️ Kurulum ve Çalıştırma

1. Projeyi bilgisayarınıza indirin.
2. Bilgisayarınızda Python yüklü olduğundan emin olun.
3. Terminal veya IDE üzerinden şu komutla çalıştırın:
   ```bash
   python rock_paper_scissors.py
